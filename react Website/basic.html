React Native uses a combination of Javascript and Html, Xml. It uses the flexibility of Javascript with the user-friendly Html.

It uses components to function.They can be predefined or custom-made. Each component has a certain set of properties called props. Some examples of components are button, text, touchable opacity etc.

Props of Button Description
onPress Call the handler when user clicks the button.
title Display the text inside the button.
accessibility Display the text for blindness accessibilty features.
color Set the background color of the Android button or set the color of iOS text.
disabled It disables all interactions for this component, if true.
textID Used to locate this view in end-to-end tests.
Props of text Description
fontSize Sets the size of the text
fontStyle Sets the styling of the text
fontWeight Specifies font weight. The values 'normal' and 'bold' are supported for most fonts. Not all fonts have a variant for each of the numeric values, in that case the closest one is chosen.
lineHeight Sets the styling of the text
textAlign Specifies text alignment. The value 'justify' is only supported on iOS and fallbacks to left on Android.
textDecorationLine Sets the styling of the text
fontFamily Sets the styling of the text
includeFontPadding Set to false to remove extra font padding intended to make space for certain ascenders / descenders. With some fonts, this padding can make text look slightly misaligned when centered vertically. For best results also set textAlignVertical to center. Default is true.
letterSpacing The letter-spacing property controls the amount of space between each letter in a given element or block of text.
textTransform The text-transform CSS property specifies how to capitalize an element's text. It can be used to make text appear in all-uppercase or all-lowercase, or with each word capitalized.
Touchable Opacity Props Description
accessible When true, indicates that the view is an accessibility element. By default, all the touchable elements are accessible.
disabled If true, disable all interactions for this component.
onFocus Invoked when the item receives focus.
onLongPress Called if the time after onPressIn lasts longer than 370 milliseconds. This time period can be customized with delayLongPress.
onPress Called when the touch is released, but not if cancelled (e.g. by a scroll that steals the responder lock). The first function argument is an event in form of PressEvent.
testID Used to locate this view in end-to-end tests.
touchSoundDisabled If true, doesn't play a system sound on touch.
