React Native is an open-source mobile application framework created by Facebook. It is used to develop applications for Android, iOS, Web and UWP by enabling developers to use React along swith native platform capabilities. It uses a combination of JavaScript and Xml.

Quiz Buzzer App

Quiz Master App

Quizes are fun competitions that teach us a lot as well, however they have some flaws, the buzzers in rapid-fire rounds are expensive and hard to maintain. These apps are the solution to this problem. Both of these apps work in tandom, a team can join in and once they press the buzzer, it will be recorded on the Quiz master app.

Both of these apps were created using the React-native platform and use the Firebase database to store information.
